dos commands

1. clear the screen : cls
2. creating a directory : mkdir kowshik
3. hide a folder : attrib +h kowshik
4. unhide a folder : attrib -h kowshik
5. remove one empty directory : rmdir kowshik
6. remove directory with contenet : rmdir /s/q kowshik
7. display the content of a directory : dir 
8. display the hidden content in directory : dir /a:h
9. rename a folder : rename old_name new_name
10. diplay the content of the file : type filename.txt
11. to create a new file : notepad filename.txt
12. to forward the content of one file to another file : type old_file>new file
13. to delete a file : del file_name
14. to delete all txt files : del *.txt
15. the list of wild charactor
  ?- single character
  *- 0 or more characters
16. display all single character named file with extension .txt:dir?.txt
17. display all files with any extension : dir *.*
18. to merge all file content : type *.txt > merged.txt
